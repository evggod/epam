Тесты проводятся на сайте www.bahn.com/

Тест 1: Проверка регистрации пользователя с пустыми полями

Шаги: открыть форму регистрации, оставить все поля пустыми, нажать на кнопку "Activate registration"
Ожидаемый результат: Пользователь не должен быть создан, выделяются поля, обязательные для заполнения. 

Тест 2: Установка даты отправления меньше текущей даты

Шаги: открыть форму бронирования, заполнить форму с заведомо ложной датой, нажать кнопку "Search"
Ожидаемый результат: Поиск не должен пройти успешно, кроме того, дата должна редактироваться еще да нажатия на кнопку. 

Тест 3: Установка даты отправления из будущего (больше 6 месяцев)

Шаги: открыть форму бронирования, заполнить поля from/to, указать заведомо ложную дату, нажать кнопку "Search"
Ожидаемый результат, Поиск не должен пройти успешно, должно появиться сообщение от максимально допустимой дате. 

Тест 4: Установка одного места отправления и прибытия. 

Шаги: открыть форму бронирования, заполнить поля from/to одинаковыми значениями, выбрать корректную дату, нажать кнопку "Search"
Ожидаемый результат: Поиск не должен пройти успешно, должно появиться предупреждение или сообщение об ошибке. 

Тест 5: Заказ билетов без ввода личных данных

Шаги: заполнить данные о заказе, при появлении окна, где требуется указать личные данные, игнорировать его
Ожидаемый результат: Бронь не должна пройти успешно. 


Тест 6: Попытка регистрации с некорректным вводом e-mail

Шаги: перейти на форму регистрации, заполнить обязательные поля, e-mail написать некорректно
Ожидаемый результат: Регистрация не должна пройти успешно, ошибочное поле должно привлечь на себя внимание и указать проблему. 

Тест 7: Попытка регистрации с вводом юзернейма на кириллице

Шаги: перейти заполнить обязательные поля регистрации, юзернейм написать на кириллице
Ожидаемый результат: Регистрация не должна пройти успешно, поле должно привлечь внимание и указать допустимые для ввода символы. 

Тест 8: Отмена бронирования

Шаги: оформить бронь, перейти в раздел "My bookings", найти необходимую бронь, попытаться отменить. 
Ожидаемый результат: Отмена должна пройти успешно, если до момента отправления достаточно времени (>1 часа).

Тест 9: Отмена бронирования в момент оплаты билета

Шаги: Оформить бронь до момента оплаты, при оплате завершить бронирование. 
Ожидаемый результат: Должно появиться всплывающее окно с повторным подтверждением отмены брони. Деньги не должны сняться со счета. 

Тест 10: Грубое завершение сеанса в момент оплаты

Шаги: Оформить бронь до момента оплаты, при оплате грубо завершить сеанс (закрыть браузер, выключить компьютер). 
Ожидаемый результат: Транзакция должна осуществиться корректно: бронь должна либо осуществиться. либо отмениться, причем в случае отмены деньги не должны списаться со счета
